### What I learned

- How to add color tint, I used this code:
```
.card-image {
    max-width: 50%;
    position: relative; 
}

.card-image:before {
    content: "";
    display: block;
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    background: rgba(170, 92, 219, 0.5);
}
```
